== Description ==
QMoney wordpress plugin for making online payment and transfer with your QMoney account. The Qmoney WordPress Plugin currently extends the WooCommerce plugin making it available at the checkout page among the list of payment options 

= Plugin/ Theme Support =

= Localization =
*English (default)*

== Installation ==
1. Make sure you have WooCommerce installed
2. Install and activate the QMoney plugin from WooCommerce Settings / Payments
3. Enable the QMoney plugin from your WooCommerce Payments
4. Click on Manage
5. Fill in your issued merchant API KEY
6. click save changes 
7. Your API KEY will be saved
